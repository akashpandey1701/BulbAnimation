# Bulb Animation

A simple Android animation built with **Jetpack Compose Canvas** that recreates a hanging light bulb with smooth interactive effects.

## Features

* Hanging wire and bulb design drawn using Canvas
* Smooth bulb animation
* Custom shapes created with `Path`
* Jetpack Compose-based implementation
* Lightweight and fully customizable

## Tech Stack

* Kotlin
* Jetpack Compose
* Compose Canvas

## Getting Started

1. Clone the repository.
2. Open the project in Android Studio.
3. Sync Gradle and run the app.

## Purpose

This project was built to explore custom drawing and animations using Jetpack Compose Canvas, focusing on creating UI elements without relying on image assets.
